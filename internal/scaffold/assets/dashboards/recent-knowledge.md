# Recent Knowledge

> Generated by AXON. Requires the Dataview plugin. Sources are ingested into
> `03-Resources/Knowledge/` by `axon ingest`.

```dataview
TABLE source_url AS "source", source_date AS "date", file.mtime AS "ingested"
FROM "03-Resources/Knowledge"
WHERE type = "source"
SORT file.mtime DESC
LIMIT 25
```

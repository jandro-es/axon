# Link Suggestions

> Generated by AXON. The `link-suggester` and `inbox-triage` automations write
> ranked proposals to `[[.axon/review-queue]]`; approve the good ones, then apply
> them with the `vault_move`/`vault_patch` MCP tools or by editing the notes.

```dataview
FROM "03-Resources" OR "01-Projects" OR "02-Areas"
WHERE length(file.outlinks) = 0

```

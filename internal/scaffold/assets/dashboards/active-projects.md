# Active Projects

> Generated by AXON. Requires the Dataview plugin.

```dataview
TABLE status, updated, file.mtime AS "last touched"
FROM "01-Projects"
WHERE type = "project" AND (status = "active" OR status = null)
SORT file.mtime DESC
```

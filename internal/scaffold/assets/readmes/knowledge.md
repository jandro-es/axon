# 03 — Resources / Knowledge

Ingested sources live here — one note per source, created by `axon ingest`.

Each note carries source provenance in its frontmatter (`source_url`,
`source_author`, `source_date`), an AXON-maintained summary inside an
`axon:summary` managed block, tags and suggested links. **Do not hand-edit** the
managed blocks or the `content_hash` field; everything else is yours.
